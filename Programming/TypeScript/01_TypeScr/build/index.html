What is Ts?

TypeScript is a superset of JavaScript that adds static typing and additional features to JavaScript.
Oue basic folder Structure when using typescript
  • src
    • main.ts (and all other ts files)
  • build
    • index.html
    • js(so we have to change tsconfig file to point to build folder , basically to tell typescript to compile js to build/js)
      • main.js
        • --- other js files (that are compiled from ts)
    • Any other build files(like css file or etc)
