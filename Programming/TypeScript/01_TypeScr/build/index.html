What is Ts?

TypeScript is a superset of JavaScript that adds static typing and additional features to JavaScript.
Oue basic folder Structure when using typescript
  • src
    • main.ts (and all other ts files)
  • build
    • index.html
    • js(so we have to change tsconfig file to point to build folder , basically to tell typescript to compile js to build/js)
      • main.js
        • --- other js files (that are compiled from ts)
    • Any other build files(like css file or etc)

Some important tsconfig file properties , for create tsconfig file run tsc --init
  • outDir - The directory where the compiled JavaScript files will be saved.
  • outFile - The file where all TypeScript code will be compiled to. But for module property have to be AMD or System and "target": "ES5"
  • rootDir - The directory that contains where all your ts files will be look by tsconfig , when you use command tsc or tsc -w
  • target- The ECMAScript target version. Basically in what version we want to compile our code
  • module - The module system to use.
  • moduleResolution - The module resolution strategy to use.
  • allowJs - Allow JavaScript files to be compiled.
  • noEmitOnError - See this clip
